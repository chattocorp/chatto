## AttachmentService

Reads room attachments and refreshes expiring signed attachment URLs.

<a id="chatto-api-v1-AttachmentService-ListRoomAttachments"></a>

### ListRoomAttachments

Lists current message-owned room attachments. Authentication and room
membership are required. Returns PERMISSION_DENIED when the room is
inaccessible to the caller.

```http
POST /api/connect/chatto.api.v1.AttachmentService/ListRoomAttachments
```

<a id="chatto-api-v1-AttachmentService-RefreshMessageAttachmentUrls"></a>

### RefreshMessageAttachmentUrls

Refreshes signed URLs for the current attachments on one message.
Authentication and room membership are required. Returns NOT_FOUND when the
message does not exist, is not a current visible message, or belongs to a
different room. Returns an empty attachment list when the message exists
but currently has no attachments.

```http
POST /api/connect/chatto.api.v1.AttachmentService/RefreshMessageAttachmentUrls
```

<a id="chatto-api-v1-RefreshMessageAttachmentUrlsRequest"></a>

#### Input: RefreshMessageAttachmentUrlsRequest

Request to refresh signed attachment URLs for one message.

| Field | Type | Description |
| --- | --- | --- |
| `room_id` | `string` | Required room ID. |
| `event_id` | `string` | Required message event ID. |
| `thumbnail` | [`AttachmentThumbnailOptions`](/reference/connectrpc-api/types/#chatto-api-v1-AttachmentThumbnailOptions) | Thumbnail URL options. Defaults are applied when absent. |


<a id="chatto-api-v1-RefreshMessageAttachmentUrlsResponse"></a>

<a id="chatto-api-v1-AttachmentService-BatchRefreshMessageAttachmentUrls"></a>

### BatchRefreshMessageAttachmentUrls

Refreshes signed URLs for the current attachments on multiple messages in
one room. Authentication and room membership are required. Missing,
retracted, non-message, and wrong-room event IDs are omitted. Results
preserve first-seen request order and repeated event IDs are de-duplicated.
Existing visible messages with no current attachments are returned with an
empty attachment list.

```http
POST /api/connect/chatto.api.v1.AttachmentService/BatchRefreshMessageAttachmentUrls
```

<a id="chatto-api-v1-BatchRefreshMessageAttachmentUrlsRequest"></a>

#### Input: BatchRefreshMessageAttachmentUrlsRequest

| `room_id` | `string` | Required room ID. |
| `event_ids` | `repeated string` | Required message event IDs. Missing, retracted, non-message, and wrong-room event IDs are omitted from the response. |
| `thumbnail` | [`AttachmentThumbnailOptions`](/reference/connectrpc-api/types/#chatto-api-v1-AttachmentThumbnailOptions) | Thumbnail URL options. Defaults are applied when absent. |
