## PersonalDataService

PersonalDataService stores the authenticated user's portable preferences and
server directory on their chosen home server.

<a id="chatto-personaldata-api-v1-PersonalDataService-GetPreferences"></a>

### GetPreferences

Gets portable preferences for the authenticated user.

```http
POST /api/connect/chatto.personaldata.api.v1.PersonalDataService/GetPreferences
```

<a id="chatto-personaldata-api-v1-GetPreferencesRequest"></a>

#### Input: GetPreferencesRequest

GetPreferencesRequest selects the authenticated user's preferences.


<a id="chatto-personaldata-api-v1-GetPreferencesResponse"></a>

#### Result: GetPreferencesResponse

GetPreferencesResponse returns the authenticated user's portable preferences.

| Field | Type | Description |
| --- | --- | --- |
