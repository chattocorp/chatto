## Key Features

- **Own your deployment**: run one binary with embedded NATS, use Docker Compose for a small production server, or scale out with Kubernetes, external NATS, S3-compatible storage, and Prometheus metrics.
- **Modern chat behavior out of the box**: channel rooms, DMs, threads, replies, reactions, mentions, typing indicators, presence modes, custom statuses, read state, and a keyboard-first quick switcher.
- **Operator-shaped communities**: room groups, sidebar links, universal announcement rooms, custom roles, server/group/room permissions, user overrides, permission inspection, server branding, and a practical admin dashboard.
- **Rich media and calls**: attachments, image sizing, room files, storage-backed link previews, optional video and animated-GIF processing, plus voice and video calls with screen sharing.
- **Notifications that work like an app**: persistent notification center, per-room levels, thread follow notifications, DND-aware alerting, installed-PWA badges, and optional Web Push with self-managed VAPID keys.
- **Privacy-first operations**: encrypted message bodies and durable user PII, crypto-shredding for account deletion, encrypted backups, key-export workflows, and an explicit boundary between operating the server and reading user content.
- **Typed integration surface**: ConnectRPC public and admin APIs, protobuf realtime WebSocket delivery, generated API reference, and a local operator socket for trusted recovery automation.
