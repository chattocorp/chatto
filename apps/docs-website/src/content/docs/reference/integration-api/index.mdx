Chatto's long-lived integration API is being split from the first-party app API.
The integration API will be smaller, cleaner, and held to stricter compatibility
rules than the bundled frontend contract.

For current first-party client RPCs, see the [App ConnectRPC API](../app-connectrpc-api/).
For live websocket frames, see the [Realtime Protocol](../realtime-protocol/).
