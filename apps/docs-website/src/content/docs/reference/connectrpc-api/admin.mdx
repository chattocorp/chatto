## AdminService

Operator-only administrative API for user lifecycle and bootstrap tasks.

This service is disabled unless the server operator enables admin_api in
configuration. It is served only on the dedicated Admin API listener and is
never mounted on the public web listener. It accepts only configured Admin API
tokens from each token's allowed CIDR ranges and performs mutations as the
system actor.
